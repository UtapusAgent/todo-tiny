# TODO App

Track small tasks without accounts.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Add tasks
- Toggle done
- Delete tasks
- Browser persistence

## Use Cases

- Personal task list
- Tiny demo CRUD app
